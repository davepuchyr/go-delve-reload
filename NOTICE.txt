backend inspired by ardanlabs service training

https://github.com/ardanlabs/service-training
https://education.ardanlabs.com/

strategies include:

- graceful shutdown
- seeding & migrations
- package oriented design
- error handling
- middleware
- cancellation
- request validation
- request logging
- self shutdown

significant changes include:

- removed original tracing implementation
- removed original auth implementation
- added seeding and migrations with go-migrate
- added integration tests using testcontainers-go
- added multi-stage dockerfiles
- added docker secrets opt-in
- added self-signed certificates
- added cors library
- added makefile utility
- added frontend application
- added custom docker-compose file
- added live reload opt-in
- added delve debugging opt-in
- added pgcli opt-in
